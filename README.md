# Ai00-X Releases

This repository hosts GitHub Releases assets for the Ai00-X Tauri desktop app (Updater). Upload installer files, corresponding `.sig` signatures, and `latest.json` generated during `pnpm tauri build`.

- Installers: `*.msi`, `*.exe`, etc.
- Signatures: matching `.sig` files
- Manifest: `latest.json` (Tauri updater manifest)

Do not commit private keys. Keep signing keys in local secure storage only.
